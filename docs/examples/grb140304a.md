# GRB 140304A: Forward Shock Energy Injection

This example demonstrates the `magnetar` energy injection model by reproducing the multi-wavelength afterglow of GRB 140304A from Laskar et al. (2018, ApJ, 859, 134; arXiv:1707.05784). The burst occurred at high redshift ($z = 5.283$) and shows clear evidence for energy injection into the forward shock between 0.15--0.26 d.

## Background

GRB 140304A was detected by Swift/BAT on 2014 March 4. The X-ray afterglow shows a dramatic re-brightening at ~0.26 d, interpreted by Laskar et al. as energy injection ($E \propto t^{3.8}$) from a shell collision. The radio SED evolves through multiple spectral components over ~90 days of VLA monitoring.

Key references:

- Laskar et al. 2018, ApJ, 859, 134 (arXiv:1707.05784) --- Multi-frequency analysis, wind model with energy injection

## Physical parameters

The paper's preferred model is a **stellar wind medium** with the spectral ordering $\nu_\mathrm{NIR} < \nu_c < \nu_X$ at 0.1 d.

| Parameter | Value | Notes |
|-----------|-------|-------|
| $E_{K,\mathrm{iso}}$ | $4.9 \times 10^{54}$ erg | Post-injection isotropic kinetic energy |
| $E_{K,\mathrm{iso,pre}}$ | $6.1 \times 10^{52}$ erg | Pre-injection energy (at 0.15 d) |
| $\Gamma_0$ | 300 | Initial Lorentz factor |
| $A_*$ | 0.026 | Wind density parameter |
| $p$ | 2.59 | Electron spectral index |
| $\varepsilon_e$ | 0.025 | Electron energy fraction |
| $\varepsilon_B$ | 0.059 | Magnetic field energy fraction |
| $\theta_j$ | 1.13 deg | Jet opening angle |
| $t_\mathrm{jet}$ | 10.6 d | Jet break time |
| $z$ | 5.283 | Redshift |

### Energy injection

The paper models the injection as a violent shell collision ($E \propto t^{3.8}$) between 0.15 and 0.26 d. In jetsimpy-rs, we approximate this using the magnetar spin-down model:

$$L(t) = L_0 \left(1 + \frac{t}{t_0}\right)^{-q}$$

which injects energy continuously into the forward shock. We use $L_0 = 10^{49}$ erg/s and $t_0 = 5 \times 10^4$ s with $q = 2$ (magnetic dipole), giving an asymptotic injected energy of $E_\mathrm{inj} = L_0 t_0 = 5 \times 10^{53}$ erg (~9x the pre-injection energy). This is deliberately moderate to avoid overwhelming the early-time dynamics, since the magnetar model injects from $t = 0$ (unlike the paper's shell collision which activates at 0.15 d).

### Density translation

The standard wind density is $n(r) = A_* \times 3 \times 10^{35} / r^2$ cm$^{-3}$. In jetsimpy-rs, $n(r) = n_\mathrm{wind} \cdot (r / 10^{17}\,\mathrm{cm})^{-2}$, so:

$$n_\mathrm{wind} = A_* \times 29.9 = 0.026 \times 29.9 \approx 0.78$$

## Computing the model

Since this is a wind medium ($k = 2$), we use the `Jet` class directly rather than the `FluxDensity_tophat` shortcut:

```python
import numpy as np
import matplotlib.pyplot as plt
from jetsimpy_rs import Jet, TopHat, ForwardJetRes

# ── Cosmology: luminosity distance for z=5.283 ──
z = 5.283
H0_cgs = 68.0 * 1e5 / 3.086e24  # H0 = 68 km/s/Mpc in s^-1
Om, OL = 0.31, 0.69
zz = np.linspace(0, z, 10000)
integrand = 1.0 / np.sqrt(Om * (1 + zz)**3 + OL)
d_C = 2.998e10 / H0_cgs * np.trapezoid(integrand, zz)
d_L_Mpc = d_C * (1 + z) / 3.086e24

# ── Model parameters ──
# Wind density: A* = 0.026 → nwind = A* × 29.9
A_star = 0.026
nwind = A_star * 29.9  # ≈ 0.78
theta_c = 0.0197  # 1.13 deg

Eiso_pre = 6.1e52   # pre-injection kinetic energy
Eiso_post = 4.9e54  # post-injection kinetic energy

P = dict(
    Eiso=Eiso_pre, lf=300, theta_c=theta_c,
    A=nwind, n0=0.0, p=2.59,
    eps_e=0.025, eps_b=0.059,
    theta_v=0.0, z=z, d=d_L_Mpc,
)

# Magnetar injection: moderate L0 to avoid overwhelming early-time dynamics
# E_inj(inf) = L0 * t0 = 1e49 * 5e4 = 5e53 erg → total ~5.6e53 (~9x E0)
mag_l0 = 1e49    # erg/s
mag_t0 = 5e4     # s (~0.58 d)
mag_q = 2.0      # magnetic dipole

# ── Helper: create jet with given Eiso and optional magnetar ──
def make_jet(eiso, magnetar_l0=0.0):
    return Jet(
        TopHat(theta_c, eiso, lf0=300),
        nwind, 0.0,
        tmin=10.0, tmax=1e9,
        grid=ForwardJetRes(theta_c, 33),
        tail=True, spread=False, k=2.0,
        magnetar_l0=magnetar_l0, magnetar_t0=mag_t0, magnetar_q=mag_q,
    )

jet_noinj = make_jet(Eiso_pre)           # no injection
jet_mag = make_jet(Eiso_pre, mag_l0)     # magnetar injection
jet_post = make_jet(Eiso_post)           # static post-injection E

P_post = {**P, 'Eiso': Eiso_post}

# ── Radio light curves (3-model comparison) ──
t = np.geomspace(100, 1e8, 300)
t_days = t / 86400

fig, axes = plt.subplots(1, 3, figsize=(16, 5), sharey=True)
titles = [
    r'No injection ($E_0 = 6.1 \times 10^{52}$ erg)',
    r'Magnetar ($L_0 = 10^{49}$ erg/s, $t_0 = 5 \times 10^4$ s)',
    r'Static post-injection ($E = 4.9 \times 10^{54}$ erg)',
]
jets = [jet_noinj, jet_mag, jet_post]
params_list = [P, P, P_post]

for ax, jet, title, params in zip(axes, jets, titles, params_list):
    for freq_GHz, color, label in zip([5.0, 15.0],
                                       ['#1f77b4', '#ff7f0e'],
                                       ['5 GHz', '15 GHz']):
        nu = freq_GHz * 1e9
        flux = jet.FluxDensity(t, nu, params)
        ax.loglog(t_days, flux, '-', color=color, lw=1.5, label=label)
    ax.set_xlabel('Time [days]')
    ax.set_title(title, fontsize=10)
    ax.set_xlim(0.1, 200)
    ax.set_ylim(5e-3, 2)
    ax.legend(fontsize=8)

axes[0].set_ylabel('Flux density [mJy]')
plt.tight_layout()
plt.savefig('grb140304a_radio.png', dpi=150, bbox_inches='tight')
plt.close()
```

## Notes

- The Laskar et al. model invokes a **shell collision** (discrete energy injection), which differs from the continuous magnetar spin-down model implemented in jetsimpy-rs. The magnetar model produces a smoother energy injection profile.
- The multi-component radio SEDs at 1.5 d and 8.5 d include a **reverse shock** contribution (not modeled here). Our forward-shock-only model captures the dominant spectral component.
- At $z = 5.283$, the IGM absorbs all optical emission shortward of $\sim 7700$ A (observer frame), so only NIR bands ($z'$, $Y$, $J$, $H$) detect the afterglow after $\sim$1 d.
- The jet opening angle ($\theta_j = 1.13^\circ = 0.0197$ rad) is unusually narrow, yielding a beaming-corrected energy $E_K \approx 10^{51}$ erg despite the large isotropic-equivalent energy.
